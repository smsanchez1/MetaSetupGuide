# **Sharing an Application Link from Meta Developer Hub**

This document outlines the procedure to generate and share a public application link from the **Meta Developer Hub**.

---

## **1. Access the Application Page**

1. Sign in to the **Meta Developer Hub**.
2. Navigate to the **application page** corresponding to the project to be shared.

![Application Page](https://github.com/smsanchez1/Images/blob/main/img/metastore.png?raw=true)

---

## **2. Locate the Distribution Channels**

1. Open the **Channels** tab on the application page.
2. Review the list of available **distribution channels** associated with the application.

![ Distribution Channels](https://github.com/smsanchez1/Images/blob/main/img/distribution.png?raw=true)

---

## **3. Select the Desired Distribution Channel**

Select the specific **distribution channel** intended for sharing the application build.

![Desired Distribution Channel](https://github.com/smsanchez1/Images/blob/main/img/channelsapp.png?raw=true)

---

## **4. Generate and Configure the Shareable Link**

1. Within the selected channel, open the **Settings** section.
2. Choose the option to **Generate New Link**.
3. Set the linkâ€™s visibility to **Public** to allow unrestricted access.
4. Copy and distribute the generated link as required.

![Configure the Shareable Link](https://github.com/smsanchez1/Images/blob/main/img/betasettings.png?raw=true)
