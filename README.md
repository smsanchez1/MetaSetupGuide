# **Meta Quest 2 & 3 Setup Guide**

This document provides detailed instructions for configuring a **Meta Quest 2**, **Meta Quest 3**, or **Meta Quest 3S** headset, from account creation to initial device setup and application installation.

---

## **1. Prerequisites**

Ensure the following components are available before beginning the setup:

* A **Meta Quest headset** (Quest 2 or Quest 3 series)
* **Controllers**, fully charged and ready to pair
* A **smartphone** (iOS or Android)
* Access to a **stable Wi-Fi connection**

![oshw2 demo fast](https://github.com/smsanchez1/Images/blob/main/metaquest.jpg?raw=true)

---

## **2. Installation of the Meta Quest Mobile Application**

1. Open the **App Store** (for iOS) or **Google Play Store** (for Android).
2. Search for **“Meta Quest”**.
3. Select **Install** or **Download**.
4. Once installed, launch the **Meta Quest** application.

![oshw2 demo fast](https://github.com/smsanchez1/Images/blob/main/metaquest.jpg?raw=true)

---

## **3. Meta Account Setup or Login**

1. Open the **Meta Quest** application.
2. Select **Continue**.
3. Choose one of the following options:

   * **Create Account** – to register a new Meta account
   * **Log In** – to sign in with an existing account
4. When creating a new account:

   * Registration may be completed using an **email address**, **Facebook**, or **Instagram** credentials.
   * Verify the email address if prompted.
   * Configure a **username**, **profile picture**, and **privacy settings**.
5. After completing registration, access to the Meta Quest app will be enabled.

> **Note:** Account creation and management can also be performed through the official portal: [https://meta.com/setup](https://meta.com/setup).

If access to **ADT account** is required, contact the designated system administrator.

![oshw2 demo fast](https://github.com/smsanchez1/Images/blob/main/metaquest.jpg?raw=true)

---

## **4. Headset Preparation and Power-On**

1. Ensure the headset is **fully charged**:

   * Connect the device to the charger until the indicator light turns **green**, signaling full charge.
2. Press and hold the **Power button** (located on the right side of the headset) until the **Meta logo** appears.
3. Put on the headset to begin the initial configuration.

The setup process will include:

* Language selection
* Wi-Fi connection setup
* Controller pairing (typically automatic)

Follow the on-screen instructions to proceed.

![oshw2 demo fast](https://github.com/smsanchez1/Images/blob/main/metaquest.jpg?raw=true)

---

## **5. Pairing the Headset with the Meta Quest App**

1. Open the **Meta Quest** application on the mobile device.
2. Log in using the same Meta account credentials created or used earlier.
3. Select **Set up new headset** when prompted.
4. Choose the correct model (**Meta Quest 2** or **Meta Quest 3**).
5. Follow the guided setup steps:

   * Ensure **Bluetooth** is enabled on the smartphone.
   * Verify the headset is **powered on and within range**.
   * Enter the **pairing code** displayed inside the headset.

Upon successful pairing, the headset will connect to the mobile device via Wi-Fi.

![oshw2 demo fast](https://github.com/smsanchez1/Images/blob/main/metaquest.jpg?raw=true)

---

## **6. Completing Headset Configuration**

Continue configuration inside the headset:

1. **Connect to Wi-Fi** if not already connected.
2. Allow the headset to **install firmware updates** if prompted.
3. Configure **Guardian boundaries**:

   * Use the 3D interface to define a **safe play area**.
   * Select **Stationary** mode for seated use.
   * Select **Roomscale** mode for active movement.
4. (Optional) Activate **hand tracking** features if supported.

   * Meta Quest 3 provides enhanced hand tracking by default.

## **8. Troubleshooting**

| Issue                          | Recommended Action                                                              |
| ------------------------------ | ------------------------------------------------------------------------------- |
| Headset not pairing            | Verify Bluetooth and Wi-Fi are enabled. Restart both headset and mobile device. |
| Controllers not tracking       | Replace controller batteries or re-pair them using the device settings.         |
| Screen flickering or black     | Hold the Power button for approximately 10 seconds to restart the headset.      |
| Application installation fails | Confirm Wi-Fi stability, available storage space, and system firmware version.  |
